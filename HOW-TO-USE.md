# 🎮 如何使用 Entropy Reverse 游戏

## 🚀 快速开始

### 1. 启动服务器

```bash
npm run dev
```

服务器将在 http://localhost:3000 启动

### 2. 打开游戏

在浏览器中访问：
- **首页**: http://localhost:3000
- **游戏页面**: http://localhost:3000/deconstruct

## 📸 游戏流程

### 步骤 1: 上传图片

1. 点击"选择图片"按钮
2. 从你的电脑选择一张物体的照片
   - 可以是任何东西：手机、杯子、汽车、玩具等
   - 图片越清晰，识别效果越好
3. 图片会显示预览

### 步骤 2: AI 识别物体

1. 点击"🔍 识别物体"按钮
2. AI 会分析图片并识别出物体
3. 你会看到：
   - **物体名称**: 具体的名字（比如"iPhone 15 Pro"而不只是"手机"）
   - **分类**: 物体的类型（电子产品、交通工具等）
   - **详细描述**: 2-3句话的详细说明

### 步骤 3: 开始拆解

1. 点击"🔨 开始拆解"按钮
2. AI 会自动递归拆解物体
3. 拆解过程：
   - 📦 **蓝色框** = 可以继续拆解的部件
   - 🌿 **绿色框** = 自然材料（拆解终点）
4. 等待1-2分钟，完整的拆解树就会显示出来

## 💡 使用技巧

### 拍照建议

1. **清晰度**: 确保图片清晰，光线充足
2. **单一物体**: 图片中最好只有一个主要物体
3. **完整性**: 尽量拍摄完整的物体，不要只拍一部分
4. **背景**: 简单的背景更容易识别

### 适合拆解的物体

✅ **推荐**:
- 电子产品（手机、电脑、相机）
- 日常用品（杯子、笔、书包）
- 交通工具（自行车、汽车）
- 家具（椅子、桌子）
- 玩具（积木、玩具车）

❌ **不推荐**:
- 食物（拆解结果可能不准确）
- 动物或植物（它们本身就是自然物）
- 抽象的东西（云、影子）

## 🎨 界面说明

### 颜色含义

- **蓝色边框** (📦): 人造部件，可以继续拆解
- **绿色边框** (🌿): 自然材料，拆解终点
- **紫色按钮**: 主要操作按钮

### 图标含义

- 📸 上传图片
- 🔍 识别物体
- 🔨 开始拆解
- 📦 可拆解部件
- 🌿 自然材料
- ✅ 拆解完成

## ⚠️ 常见问题

### Q: 识别失败怎么办？

A:
1. 检查图片是否清晰
2. 尝试换一个角度拍摄
3. 确保物体在图片中占主要位置
4. 检查网络连接和 API Key 配置

### Q: 拆解时间太长？

A:
- 复杂物体（如手机、汽车）可能需要1-2分钟
- 简单物体（如杯子、笔）通常30秒内完成
- 这是正常的，AI 需要时间递归分析每个部件

### Q: 拆解结果不准确？

A:
- AI 的拆解基于通用知识，可能不是100%准确
- 这是一个教育游戏，重点是理解物体的组成
- 如果发现明显错误，可以尝试重新拆解

### Q: 可以拆解多个物体吗？

A:
- 目前每次只能拆解一个物体
- 拆解完成后，可以上传新图片开始新的拆解

## 🎓 教育价值

这个游戏可以帮助孩子：

1. **理解物体构成**: 了解日常物品是如何制造的
2. **认识自然材料**: 知道哪些材料来自大自然
3. **培养好奇心**: 激发对科学和工程的兴趣
4. **系统思维**: 学习如何将复杂事物分解为简单部分

## 🔧 技术支持

如果遇到问题：
1. 查看浏览器控制台（F12）的错误信息
2. 检查服务器终端的日志
3. 确认 `.env.local` 文件配置正确
4. 重启开发服务器
